NLP module for identifying lung nodules.

# Requirements #
* Python 3.3+
* nltk (for tokenizer)

# Documentation #
## Install ##
* `git clone https://bitbucket.org/dcronkite/crn-lung-nodule`
* `python setup.py install`

## Usage ##
Run `crn-algorithm -h` for argument details.
